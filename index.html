记Hexo+Github搭建博客

作为一个技术小白，心血来潮想自己搭个博客玩玩，又对自己的记忆能力和更博频率很是没有信心。所以想以第一篇博文来记录下自己糊里糊涂的搭建过程，方便自己以后回顾，顺便第一次使用Markdown也算是熟悉下语法。搭建过程都是来自网上教程参考。下面开始~

Github建立仓库

登陆Github网站,点击New repository新建一个仓库，在仓库名输入username.github.io，当然username是自己的账号名啦。这里使用的是Github提供的Pages服务。

Hexo环境搭建及安装

  1. 1.下载Node.js
    用于生成静态页面
  2. 2.下载Github for windows
    用于把本地Hexo内容提交到github上去
  3. 3.安装Hexo
    首先在想要放置的位置建立一个文件夹如blog(比如我是以E盘作为本地位置)，执行如下命令安装Hexo：
    cd E:\blog
    npm install hexo-cli -g
    初始化Hexo
    hexo init
    初始化成功后，执行以下命令进行本地预览：
    hexo generate
    hexo server
    提示成功后，在浏览器地址栏输入http://localhost:4000 ，就可以预览本地博客了，如不成功可在设置文件里更改端口号试试。

将本地博客部署到Github

  1. 1.配置Github
    用文本编辑器打开blog文件夹下的_config.yml文件，把deploy标签下内容修改如下，其中repo填写自己的github仓库地址：
    type: git
    repo: https://github.com/username/username.github.io.git
    branch: master
    之后在Git Shell中输入命令：
    npm install hexo-deployer-git –save
  2. 2.之后每次部署，输入以下命令：
    hexo clean
    hexo generate
    hexo deploy
    其中generate与deploy可以组合执行：
    hexo g -d
    到这一步，通过username.github.io就可以访问自己的博客了。

##更换主题
作为外貌协会一员，默认主题兼职不能忍，看到网上大家都推荐yilia主题，果断拿来用了，感谢作者~

  1. 1.首先是下载
    在blog路径下输入命令：
    git clone http://github.com/litten/hexo-theme-yilia.git themes/yilia
  2. 2.配置
    打开blog根目录下的配置文件_config.yml，修改主题theme: yilia，顺便，还可以修改title，subtitle，description，author，language等等。
    打开themes\yilia文件下的配置文件_config.yml，可以修改主页menu，个人链接，头像url，友情链接等等。
  3. 3.部署
    依次clean generate deploy，刷新下自己的页面，是不是就修改成功了呢~

发表文章

在blog路径下输入命令：
hexo new “文章名”
之后本地blog\source_posts文件下就多了一个 文章名.md 文件，然后就可以使用Markdown编辑器来写博文了。
也可以使用新建页面命令：
hexo new page “页面名”
博文写好后，执行上述部署的三条命令，就可以把博客部署到Github上去了。

实现双电脑博客同步

刚搭完环境就想着如果换了电脑怎么同步更新博文，试着在Github仓库(主分支为master)新建了hexo分支，并把source文件夹和yilia主题配置文件push了上去。由于第一次接触git，还没搞清版本控制的一些原理知识，所以以下内容只是暂时之想，不知道实行起来如何。

每次更新完博文后并且部署完毕后，都把最新的source文件夹push到分支hexo上去。换电脑后，pull下source文件夹，替换本地sourse文件？然后再执行更新博文的步骤。
